# gt7php

Gran Turismo 7 telemetry viewer in the terminal, written in PHP.

Grabs the live telemetry GT7 broadcasts over UDP on your local network and
shows it in the terminal.

Why PHP? Why not.

## Requirements

- PHP 8.1+ (`ext-sockets`, `ext-sodium`)
- Machine on the same network as the PS4/PS5
- GT7 running

## Status

Work in progress. Right now just getting the connection working.
